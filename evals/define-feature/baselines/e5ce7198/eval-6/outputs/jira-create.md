# Jira Create Issue Parameters

```
create_issue(
  project_key: "TC",
  issue_type_id: "10142",
  summary: "Add automated PR review posting for eval results",
  description: <Feature description from preview.md>,
  labels: ["ai-generated-jira"]
)
```

## Parameters

| Parameter | Value |
|---|---|
| `project_key` | `TC` |
| `issue_type_id` | `10142` |
| `summary` | Add automated PR review posting for eval results |
| `labels` | `["ai-generated-jira"]` |
| `assignee` | *(unassigned)* |
