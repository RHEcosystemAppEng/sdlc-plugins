The feature adds a new remediation module to the backend that aggregates vulnerability remediation status from existing advisory and SBOM data. Two new REST endpoints provide summary statistics (by severity x status with 30-day trend) and per-product breakdowns. A third endpoint exports the data as CSV for management reporting (non-MVP). The frontend adds a new dashboard page with three visual components: summary cards for at-a-glance status, a progress chart for trend visualization, and a filterable table for drill-down investigation.
