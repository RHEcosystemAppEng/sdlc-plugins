## Architecture Summary

The feature introduces a side-by-side SBOM comparison capability spanning two repositories:

1. **Backend (trustify-backend)**: A new `GET /api/v2/sbom/compare?left={id1}&right={id2}` endpoint that computes a structured diff between two SBOMs on-the-fly. The diff covers six categories: added/removed packages, version changes, new/resolved vulnerabilities, and license changes. No new database tables are needed — the service queries existing `sbom_package`, `sbom_advisory`, and `package_license` join tables.

2. **Frontend (trustify-ui)**: A new comparison page at `/sbom/compare` built with PatternFly 5 components per Figma designs. The page features SBOM selectors with typeahead, collapsible diff sections with color-coded count badges, critical vulnerability highlighting, URL-shareable comparisons via query parameters, and JSON/CSV export functionality.
