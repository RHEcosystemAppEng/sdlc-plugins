### Architecture Summary

The feature adds a side-by-side SBOM comparison capability spanning two repositories:

**Backend (trustify-backend):** A new comparison module follows the existing `model/ + service/ + endpoints/` pattern. The diff is computed on-the-fly from existing package, advisory, and license data using set-based operations -- no new database tables. The `GET /api/v2/sbom/compare?left={id1}&right={id2}` endpoint returns a structured `SbomComparisonResult` with six diff categories.

**Frontend (trustify-ui):** A new SbomComparePage at `/sbom/compare` renders the comparison results using PatternFly 5 components (Select dropdowns for SBOM selection, ExpandableSection for collapsible diff sections, color-coded Badge counts, Table for data). The page supports URL-shareable comparisons via query parameters and uses virtualized lists for large diffs (>100 items). The SBOM list page gains checkbox selection and a "Compare selected" button.
