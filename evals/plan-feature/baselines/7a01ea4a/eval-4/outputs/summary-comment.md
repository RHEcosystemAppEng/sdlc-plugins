### Architecture Summary

The license compliance report feature adds a new GET /api/v2/sbom/{id}/license-report endpoint that returns packages grouped by license type with compliance flags. The implementation follows the established module pattern (model/service/endpoints) within the existing SBOM module (modules/fundamental/src/sbom/). A configurable license policy (JSON file in config/) determines which licenses are compliant. The service aggregates data from existing package_license and sbom_package entities without creating new database tables. Transitive dependencies are included by walking the full dependency tree.
