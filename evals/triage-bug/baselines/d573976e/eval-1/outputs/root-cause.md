## Root Cause

The plan-feature skill's convention conformance analysis extracts `CONVENTIONS.md` heading names
using `line[3:]`, which captures everything after the `## ` prefix but does not strip trailing
whitespace. When a heading line contains trailing spaces (e.g., `## Migration Patterns  `), the
extracted section name retains those spaces (`"Migration Patterns  "`). The subsequent
convention-aware task enrichment step performs an exact string match against expected convention
names (without trailing spaces), causing the lookup to fail silently. No warning or error is
logged, so the convention is simply omitted from the generated task description.

## Affected Files

- `plugins/sdlc-workflow/skills/plan-feature/SKILL.md` — convention heading extraction logic
  (`section_name = line[3:]` without `.strip()` or `.rstrip()`)
- `plugins/sdlc-workflow/skills/plan-feature/SKILL.md` — convention-aware task enrichment lookup
  (`if convention_name in discovered_conventions`)

## Suggested Approach

Apply `.strip()` (or `.rstrip()`) to the extracted section name at the point of extraction
(`line[3:].strip()`), ensuring trailing whitespace is normalized before the heading name is
stored as a dictionary key. This is a single-line fix at the extraction site. The lookup code
does not need to change — normalizing at the source ensures all downstream comparisons work
correctly.

Additionally, consider logging a warning when a convention heading is encountered with
non-trivial trailing whitespace, to surface formatting issues in `CONVENTIONS.md` files
to users.

## Reproducer Strategy

1. Create a `CONVENTIONS.md` fixture file with a heading that has trailing whitespace
   (e.g., `## Migration Patterns  ` with two trailing spaces).
2. Run the plan-feature convention conformance analysis against this fixture.
3. Assert that the convention IS included in the generated task's Implementation Notes
   (expected: `Per CONVENTIONS.md Migration Patterns: ...`).
4. Before the fix, this test should fail (convention is silently dropped).
   After the fix, the test should pass (convention is correctly matched despite trailing whitespace).
