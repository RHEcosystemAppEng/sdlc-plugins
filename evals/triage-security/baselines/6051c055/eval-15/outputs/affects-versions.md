## Correction Details

The PSIRT-assigned Affects Versions is incorrect. The issue is scoped to stream 2.2.x (per summary suffix `[rhtpa-2.2]`), so only 2.2.x versions are included.

- **Current**: RHTPA 2.0.0
- **Proposed**: RHTPA 2.2.0, RHTPA 2.2.1, RHTPA 2.2.2

RHTPA 2.0.0 does not correspond to any version in the supportability matrix. The affected 2.2.x versions are 2.2.0, 2.2.1, and 2.2.2 based on lock file analysis (quinn-proto < 0.11.14). Versions 2.2.3 and 2.2.4 are not affected (they ship quinn-proto 0.11.14).
