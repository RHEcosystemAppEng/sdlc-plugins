## Rationale

The cross-CVE overlap detection in Step 4.3 is designed to prevent duplicate remediation work. When two CVEs affect the same upstream component in the same stream, and one CVE's remediation task already bumps the library past the other CVE's fix threshold, the second CVE does not require its own remediation task. The version bump in TC-8009 (axios 1.7.4 -> 1.9.0) resolves both:

- **CVE-2026-42035** (Prototype Pollution via header parsing): fix requires axios >= 1.8.0
- **CVE-2026-44492** (SSRF via crafted URL): fix requires axios >= 1.8.2

Since 1.9.0 satisfies both thresholds, TC-8009 is the single remediation task that covers both CVEs. Creating a separate task for TC-8010 would be redundant.
