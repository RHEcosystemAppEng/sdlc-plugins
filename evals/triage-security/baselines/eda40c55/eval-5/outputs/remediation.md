## Description

Remediate CVE-2026-40215: update openssl-libs to 3.0.7-28.el9_4.

A buffer over-read vulnerability was found in openssl-libs during X.509
certificate chain verification (CVSS 7.1 High). Versions before
3.0.7-28.el9_4 are affected. The fix adds bounds checking before reading
Subject Alternative Name extension data.

Affected versions: 2.2.0 (3.0.7-25.el9_3), 2.2.1 (3.0.7-27.el9_4),
2.2.2 (retag of 2.2.1)
Already fixed in: 2.2.3 (3.0.7-28.el9_4), 2.2.4 (3.0.7-28.el9_4)

Advisory: https://access.redhat.com/errata/RHSA-2026:4021
CVE record: https://www.cve.org/CVERecord?id=CVE-2026-40215

Note: The latest released versions (2.2.3, 2.2.4) already ship the fixed
openssl-libs version. This task ensures the fix is documented and tracked,
and that rpms.lock.yaml continues to pin the fixed version for future builds.
