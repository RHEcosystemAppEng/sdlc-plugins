## 3.2 -- Compare and Correct Affects Versions

**Current Affects Versions (PSIRT-assigned)**: `[RHTPA 2.0.0]`
**Problem**: RHTPA 2.0.0 does not exist in the supportability matrix -- PSIRT assigned an incorrect version.

**Version impact table (scoped to stream 2.2.x only)**:

| Version | Affected? |
|---------|-----------|
| RHTPA 2.2.0 | YES (quinn-proto 0.11.9) |
| RHTPA 2.2.1 | YES (quinn-proto 0.11.12) |
| RHTPA 2.2.2 | YES (retag of 2.2.1) |
| RHTPA 2.2.3 | NO (quinn-proto 0.11.14) |
| RHTPA 2.2.4 | NO (quinn-proto 0.11.14) |

**Proposed correction**:
```
Current: [RHTPA 2.0.0] -> Proposed: [RHTPA 2.2.0, RHTPA 2.2.1, RHTPA 2.2.2]
```
